#cinescope-movie Web App

##  Project Description
This project is a Movie Explorer web application built using HTML, CSS, and JavaScript. It allows users to search for movies and view details using a public API.

##  API Used
OMDb API (Open Movie Database)
https://www.omdbapi.com/

##  Features
- Search movies by name
- Display movie posters and details
- Responsive UI design
- API integration using fetch

##  Technologies Used
- HTML
- CSS
- JavaScript
- OMDb API

##  How to Run
1. Clone the repository
2. Open folder in VS Code
3. Add your API key in script.js
4. Open index.html in browser
