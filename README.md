
🎬 CineScope – Movie Explorer App

📌 Overview

CineScope is a web-based movie exploration application that allows users to discover and browse movies using real-time data from The Movie Database (TMDB) API.

🎯 Purpose

The goal of this project is to demonstrate API integration, search functionality, and dynamic data handling using modern frontend technologies.

🚀 Features
	•	🔍 Search movies by title
	•	🎭 Filter movies by genre
	•	⭐ Sort movies by rating and popularity
	•	📅 Filter by release year
	•	📄 View detailed movie information

🔗 API Used
	•	TMDB (The Movie Database API)

🛠️ Technologies Used
	•	React.js
	•	JavaScript (ES6+)
	•	Vite
	•	CSS / Tailwind CSS
	•	REST APIs
